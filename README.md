# Autorun-Vaccine

## 사용법
감염된 USB를 컴퓨터에 꽂은 후 이 프로그램을 실행합니다.
왠만하면 [완전 삭제]옵션을 고르는게 호환성이 좋습니다.
완전 삭제 옵션을 선택하면 바이러스가 검역소에 보관되지 않고 완전 제거됩니다.

## 호환성
Windows XP / 7 / 8 / 8.1에서 테스트되었습니다.
